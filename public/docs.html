FlexAPI

API Documentation

Introduction

FlexAPI is a comprehensive exercise database API that offers a vast collection of targeted exercises. The API provides a structured format for accessing exercise information through keyed objects. Each object includes details such as body part, exercise name, instructions, and targeted muscles.

Getting Started

Authentication

No authentication required. Access targeted exercise details effortlessly.

API response format

                "Chest": [
{
"name": "Push-Ups",
"instructions": "Assume a high plank position with your hands slightly wider than shoulder-width apart. Lower your body until your chest almost touches the ground, then push yourself back up to the starting position.",
"targetedMuscles": ["Chest", "Shoulders", "Triceps"]
},

Endpoints

To fetch a exercise from the API, you can make a simple GET request to the specified endpoint. The response will be a JSON object with some of the following properties:

  • bodyPart: The body part targeted by the exercise
  • exercise: The type of exercise
  • name: The name of the exercise
  • instructions: Instructions on how to preform the exercise
  • targetedMuscles: The muscles targeted by the exercise

Fetch a random exercise

                    
GET
https://localhost:8000/api/random

Fetch all exercises

                    
GET 
https://localhost:8000/api/all

Fetch all bodyParts

                    
GET 
https://localhost:8000/api/bodyParts

Fetch all exercises names

                    
GET 
https://localhost:8000/api/exercises

Fetch exercises by bodyPart

                    
GET 
https://localhost:8000/api/${bodyPart}

Attribution

When using the data obtained from the FlexAPI, it is important to provide proper attribution. Please include the following attribution statement in your application or website:

Exercise data provided by FlexAPI (https://placeholdersite.com)

By including this attribution, you contribute to giving credit to the data source and acknowledge the efforts put into providing the exercise information. It also helps promote the FlexAPI platform to others.
